З цим оновленням Orbot для iOS повинен знову стати доступним для багатьох користувачів!

Щоб зменшити використання оперативної пам'яті (і, отже, не дозволити йому бути вбитим контролером пам'яті iOS), зробіть наступне:
- Очищайте кеш Tor при кожному запуску. (Існує налаштування, яке робить це автоматично!)
- Вимкніть використання GeoIP. (Це також нове налаштування).
- На жаль, нам довелося видалити підтримку мостів (Obfs4proxy і Snowflake), які також використовують занадто багато оперативної пам'яті.
- В iOS 17.3.1, схоже, є баг з VPN. Перезавантаження пристрою може допомогти.


У версії для macOS збережено всі функції, включно з підтримкою мостів, оскільки там немає обмежень на оперативну пам'ять.

Список змін:
- Видалено підтримку Obfs4proxy і Snowflake на iOS для зменшення використання оперативної пам'яті.
- Додано перемикач для вимкнення GeoIP, який зменшує використання оперативної пам'яті на iOS.
- Оновлена залежність від листя, що використовується для маршрутизації трафіку.
- Оновлена підтримка Snowflake для macOS.
