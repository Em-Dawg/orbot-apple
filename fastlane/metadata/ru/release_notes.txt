С этим обновлением Orbot для iOS должен снова стать пригодным для использования многими людьми!

Чтобы уменьшить использование оперативной памяти (и, следовательно, не дать ей быть убитой iOS memory watchdog), сделайте следующее:
- Очищайте кэш Tor при каждом запуске. (Есть настройка, которая делает это автоматически!)
- Отключите использование GeoIP. (Это тоже новая настройка).
- К сожалению, нам пришлось удалить поддержку мостов (Obfs4proxy и Snowflake), которые также потребляют слишком много оперативной памяти.
- В iOS 17.3.1, похоже, есть ошибка с VPN. Перезагрузка устройства может помочь.


В версии для macOS остались все функции, включая поддержку мостов, так как там нет ограничений по оперативной памяти.

Changelog:
- Удалена поддержка Obfs4proxy и Snowflake на iOS для уменьшения использования оперативной памяти.
- Добавлен тумблер для отключения GeoIP, который уменьшает использование оперативной памяти на iOS.
- Обновлена зависимость leaf, используемая для маршрутизации трафика.
- Обновлена поддержка Snowflake для macOS.
